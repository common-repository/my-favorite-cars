== Description ==

Select and display a list of your favorite cars. You can select used vehicles from dealerships from the U.S., save them to your favorites list and then display them in the sidebar, page or post.

My favorite cars plugin uses CarMusing API to get your selection of vehicles. The User ID and Activation Number are used to connect to the CarMusing server.

Brought to you by [CarMusing](http://carmusing.com/).

== Installation ==

Installation is simple:

1. Download My Favorite Cars plugin
2. Upload with Wordpress
3. Activate and Sign Up / Login (After login your User ID and Activation Number are automatically saved to the WP database)
4. Add the widget in your sidebar or shortcode into your page/post
